Task Description: Tips & Tricks

To give this variable a value, you must assign that value to it. This variable assignment looks like the following:

message = “Welcome to Java!”;

The equals sign is known as the assignment operator. The variable is placed on the left-hand side, and the value is placed on the right-hand side. Also notice that you do not specify the String type. That type only ever needs to be stated when you declare the variable.
